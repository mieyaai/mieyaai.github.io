国内ChatGPT账号注册教程，无需手机验证码

发表于2024-03-01|更新于2024-03-03|gpt
|阅读量:

之前注册 ChatGPT 除了有一个海外邮箱，还需要手机号接码，一般都是推荐一些虚拟手机号接码平台

现在通通不用！

只需要你能科学上网，即可注册。

注册完后，或者你已经有账户，想要快速升级 Plus，可以看这篇教程：国内升级ChatGPT4.0的方法

注册步骤：
国内的邮箱都不再允许注册 ChatGPT，建议申请一个海外邮箱 ,如 Gmail 也可以，就是注册难度高一些，步骤繁琐。

一、注册 OpenAI 账号

  1. 首先我们需要一个邮箱，可以是Gmail,也可以是微软的邮箱等。
  2. 申请openai。

  • 首先打开OpenAI的账户注册页面，输入邮箱:

    https://chat.openai.com/

  • 然后去邮箱里验证，在邮箱里找到邮件点击「Verify email address」。
  • 提供你的姓名和生日并点击继续
  • 选择创建ChatGPT账号即可。

    Openai的用户分成两大类：分别是ChatGPT用户和API开发者用户，两个体系是相互独立的，用户名密码是一致的，但是访问入口和支付都是独立的：
    ChatGPT 用户的登录入口是 https://chat.openai.com
    API 开发者用户的登录入口是 https://platform.openai.com

ChatGPT 用户获取 GPT-4 的使用权限，是通过购买 GPT Plus 会员，会员的收费标准是 20 美元/每月。
购买 GPT Plus 会员后，可以在 Chat 界面选择 GPT4 进行聊天。
购买教程在这里:

国内升级ChatGPT4.0的方法

API 开发者用户获取 GPT-4 API的使用权限，是通过升级付费帐户，升级付费帐户的教程在这里。
然后需要预充值5美元以上，才能开通GPT-4的访问权限
